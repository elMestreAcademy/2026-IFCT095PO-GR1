# 4 pilares de la programacion estructurada

- Expresiones (operaciones)
  - Una expresion es un tipo de operacion que el PROCESADOR debe ejecutar
- Datos y tipos de datos
  - Datos:
    - asignar <- guardamos un TIPO y VALOR en un IDENTIFICADOR
    - referir <- RECUPERAMOS el DATO a traveés del IDENTIFICADOR
  - Tipos de datos:
    - int (entero) (sin y con signo)
    - float (double)
    - str (cádena de caracteres)
      - Boolean: [True, False]
      - NoneType: [None]
- Estructuras de control
  - if
  - while
  - for
- Funciones
